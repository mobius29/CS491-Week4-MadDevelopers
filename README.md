# madCamp_proj4 (Mad-Developers)

## Stack
* Frontend
  * Language: typeescript
  * IDE: VisualStudio Code
  * Library: React
* Backend
  * Language: typescript
  * IDE: VisualStudio Code & NeoVim
  * Server: Node
  * Database: Mysql

## Function
* 회원가입, 로그인, 로그아웃, 회원탈퇴, 유저 페이지
* 게시물 작성, 수정, 삭제
* 댓글, 답글 작성
* 타자 검정
